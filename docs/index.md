# OpenClaw 技术完全解析

## 一句话概述

**OpenClaw** 是一个运行在用户本地设备上的多通道 AI 网关，能够在 WhatsApp、Telegram、Slack、Discord、Signal、iMessage 等 20+ 消息平台上提供统一的 AI 助手服务。

## 核心特性

- **多通道**: 支持 20+ 消息平台
- **本地运行**: 数据存储在本地，保护隐私
- **AI 驱动**: 基于 Pi Agent Core
- **可扩展**: 35+ 官方插件
- **开源**: MIT 许可证
